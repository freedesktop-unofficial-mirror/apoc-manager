Resumen de configuración

Se trata de una vista de sólo lectura de todos los valores de configuración que contienen datos del perfil seleccionado.

La página Resumen de configuración contiene la tabla Resumen de configuración. En ella figuran los valores de configuración que contienen datos del perfil activo, agrupados por categorías. La tabla se compone de los campos siguientes:

  • Nombre: el nombre del valor

  • Valor: el valor del parámetro

  • Estado: en esta columna figuran las entradas del tipo “Definido en <profileName>, en la que <profileName> es un vínculo que pasa del Editor de perfiles a la página Valores de configuración, y busca el valor concreto dentro de este perfil.
