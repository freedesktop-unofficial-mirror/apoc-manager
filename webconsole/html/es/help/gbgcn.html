Creación de un depósito de configuración

Un depósito de configuración consiste en una ubicación de almacenamiento de la información jerárquica de la organización, además de los datos de asignaciones y perfiles de cada elemento.

Pasos

  1. En la tabla Depósitos de configuración, haga clic en Nuevo

    Se abre el Asistente para nuevo depósito de configuración. Este asistente sirve para crear un depósito de configuración que se utiliza con Desktop Manager.

  2. En el campo Nombre de depósito, asigne un nombre al nuevo depósito de configuración y haga clic en Siguiente.

  3. En la lista Tipo de depósito, seleccione la clase de depósito.

    El asistente es válido para configurar tres tipos de depósitos:

    • LDAP: La jerarquía se toma de LDAP y todos los datos se almacenan en LDAP.

    • Basado en archivos: La jerarquía se toma de un archivo y todos los datos se almacenan en directorios y archivos.

    • Híbrido: La jerarquía se toma de LDAP, pero todos los datos se almacenan en directorios y archivos.

  4. Haga clic en el botón Siguiente.

  5. Especifique los detalles del servidor LDAP que desea configurar para este depósito.

    Hace falta indicar el nombre de host y el puerto en que se ejecuta el servidor. Asimismo, se puede elegir si conectarse o no con este servidor mediante SSL.

  6. La configuración que se haya elegido en los pasos anteriores condiciona los pasos posteriores. Siga las instrucciones que proporciona el asistente.
