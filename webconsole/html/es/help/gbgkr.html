Elementos asignados

En la página Elementos asignados figuran los elementos asignados al perfil activo; en ella se asignan o anulan asignaciones de elementos.

Los elementos de perfiles activos se muestran en la tabla Elementos asignados, que presenta la información siguiente:

  • Nombre: nombre del elemento al que se asigna el perfil activo.

  • Ruta: ubicación del elemento al que se asigna el perfil activo.

Consulte Asignación de elementos a perfiles, Anulación de asignaciones de elementos de perfiles
