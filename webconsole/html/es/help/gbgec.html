Editor de perfiles

El Editor de perfiles sirve para crear y modificar perfiles. También permite ver la configuración y las opciones del perfil activo.

El Editor de perfiles se abre haciendo clic en un nombre de perfil de una de las tablas del área de contenido Perfiles — Vista de Árbol o en la página Perfiles — Todos.

Sugerencia– Parea crear un perfil, consulte Creación de perfiles.

El Editor de perfiles contiene las páginas siguientes:

  • Propiedades generales

  • Valores de configuración

  • Elementos asignados

  • Resumen de configuración

  • Opciones avanzadas
