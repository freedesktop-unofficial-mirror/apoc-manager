Visualización y edición de los valores de configuración de perfiles

Pasos

  1. Para ver el valor de configuración de un perfil, búsquelo en la lista de categorías que hay en la parte derecha de la página.

    Los valores de configuración se organizan jerárquicamente en categorías que equivalen a las opciones de configuración de aplicaciones de escritorio como StarOffice™, Mozilla™, Evolution o Gnome.

  2. Haga clic en un valor de configuración.

    Aparece en pantalla la configuración del perfil con los valores vigentes en ese momento. Si es preciso, se proporciona información contextual complementaria, por ejemplo los efectos de valores determinados o el intervalo de valores permitido.

    Nota – Si en lugar de un valor de configuración selecciona una categoría, en pantalla aparece una tabla con valores de categorías o subcategorías.

  3. En cada valor de configuración se puede establecer un valor concreto o no definirlo.

    Si opta por no definirlo, el valor de este perfil no afecta a la aplicación, que recurre al valor predeterminado que se haya proporcionado de fábrica. En general, las listas y aplicaciones del Editor de perfiles establecen el valor predeterminado mediante texto de ayuda adicional o, si es posible, con marcador directamente en la lista de opciones del valor.

    Nota – Es importante saber que varios perfiles pueden afectar al mismo usuario. Por ejemplo, se puede crear un perfil con valores para una determinada organización. Estos valores quedan, automáticamente, disponibles para todos los usuarios que pertenecen a la misma organización. Asimismo, se pueden definir perfiles adicionales con valores de configuración para suborganizaciones. Estos valores pueden sobrescribir los de la organización superior; por ejemplo, se pueden definir distintas configuraciones de proxy para suborganizaciones, sin cambiar otros valores. Así pues, la configuración que se aplica a un usuario concreto puede llegar a ser el resultado de un proceso de combinación que abarca todos los perfiles asignados a la organización, las suborganizaciones y, probablemente, el usuario.

    Sugerencia– En algunos casos, quizá desee controlar el proceso de combinación y aplicar un valor de configuración concreto, lo que significa que otros perfiles no puedan sobrescribir dicho valor y definir otro. El Editor de perfiles posibilita especificar un comportamiento de combinación concreto para cada valor. El valor predeterminado es el de que "los perfiles pueden sobrescribir" un valor. Si anula esta selección, ni el valor ni otros perfiles podrán sobrescribir su valor.
