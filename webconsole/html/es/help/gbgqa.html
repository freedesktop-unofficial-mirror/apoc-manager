Ventana Configuración de depósitos

Esta ventana es la ubicación de almacenamiento de los datos de configuración que necesita para configurar las aplicaciones, así como la ubicación de almacenamiento de los datos de la jerarquía organizativa, junto con los perfiles y las asignaciones de cada elemento. En la ventana Configuración de depósitos se pueden crear, utilizar y sincronizar depósitos.

La ventana consta de un encabezado y la lista Depósitos de configuración.

El encabezado proporciona varios vínculos generales. En la parte superior está la barra de utilidades, con los cuatro vínculos siguientes (de izquierda a derecha):

  • El botón Console abre la página de inicio de Java Web Console.

  • El botón Versión abre una ventana con la información sobre la versión de Desktop Manager.

  • Cerrar sesión sale de Java Web Console y de Configuration Manager, y devuelve al usuario a la página de inicio de sesión.

  • El botón Ayuda abre las páginas de la Ayuda en línea.

En la parte inferior del encabezado figuran los nombres del producto, del servidor y del administrador que ha iniciado la sesión.
