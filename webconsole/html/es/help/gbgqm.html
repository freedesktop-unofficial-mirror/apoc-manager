Sincronización de depósitos de configuración

Con Desktop Manager se pueden administrar en paralelo varios extremos. De esta forma, se puede definir un extremo para verificar y otro para la producción final. Los cambios de configuración se pueden comprobar primero en el extremo de verificación. A partir de entonces, se puede sincronizar partes o todo el extremo de verificación con el de producción, a fin de aplicar los cambios pertinentes con rapidez y seguridad en el sistema de producción. Este método también es válido para un mecanismo sencillo de copia de seguridad y restauración del extremo de producción. En la ventana de sincronización se pueden efectuar las acciones siguientes.

Pasos

  1. En la ventana Depósitos de configuración, seleccione el depósito que desea sincronizar.

    Se abre la ventana de sincronización.

  2. En la lista Depósito de configuración de origen, seleccione el depósito de origen pertinente.

    El depósito de origen que actúa como origen de los cambios se puede cambiar. Este depósito no se cambia.

  3. (Opcional) Se puede restringir el depósito de origen si hace clic en el botón Cambiar que hay junto a Punto de inicio de origen.

    Esta acción establece como punto de origen de la sincronización otro elemento que no es el root.

    Se abre un cuadro de diálogo para buscar el elemento de la organización o el dominio. Tras seleccionar un elemento como Punto de inicio de origen aparece la nueva ruta.

  4. En la lista Depósito de configuración de destino, elija un depósito de destino.

    El depósito de destino que actúa como destino de los cambios se puede cambiar. Este depósito se cambia.

  5. (Opcional) Se puede restringir el depósito de destino si hace clic en el botón Cambiar que hay junto a Punto de inicio de destino.

    Esta acción establece como destino de la sincronización otro elemento que no es el root.

    Se abre un cuadro de diálogo para buscar el elemento de la organización o el dominio. Tras seleccionar un elemento como Punto de inicio de destino aparece la nueva ruta.

  6. Haga clic en el botón Comparar.

    Los dos árboles se comparan; las diferencias se reflejan en una tabla.

  7. Habilite las casillas de verificación junto a las filas de la tabla con los perfiles que desea sincronizar y haga clic en el botón Sincronizar.

    Después de la sincronización, el depósito de destino coincide con el de origen respecto a los perfiles seleccionados.
