설정 요약

설정 요약은 선택한 프로필의 데이터가 포함된 모든 구성 설정의 읽기 전용 보기입니다.

설정 요약 페이지에는 설정 요약 테이블이 포함되어 있습니다. 이 테이블에는 현재 프로필의 데이터가 포함된 구성 설정을 범주별로 분류하여 표시합니다. 이 테이블은 다음 필드로 구성됩니다.

  • 이름: 설정 이름

  • 값설정 값

  • 상태: 상태 열에는 “<profileName>에 정의된" 항목 유형이 들어 있습니다. 여기서 <profileName>은 프로필 편집기를 구성 설정 페이지로 전환하고 이 프로필의 특정 설정으로 이동하는 링크입니다.
