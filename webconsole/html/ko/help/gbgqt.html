찾기 기능 사용

찾기 기능을 사용하면 확장된 요소의 직계 자식에 해당하는 모든 요소에 대한 정규화된 검색을 수행할 수 있습니다. 이 기능은 검색 기능과는 달리, 요소에 대한 반복 검색을 제공합니다.

단계

  1. 탐색 트리에서 일부 요소는 표시되지 않음 노드를 클릭합니다.

    찾기 창이 열립니다.

  2. 찾기 필드에 전체 또는 부분 문자열을 입력합니다.

    검색은 대/소문자를 구분하거나 제한하지 않습니다. 예를 들어, 검색 문자열 "bc"는 "ABC"와 "bcd"를 찾습니다. 별표(*)는 검색 문자열 내 임의의 문자 시퀀스를 나타냅니다. 검색 문자열에서 원하는 모든 위치에 두 개 이상의 별표를 사용할 수 있습니다. 빈 필드는 “*” 검색 문자열이 있는 필드와 같은 기능을 합니다.

  3. 찾기 버튼을 클릭합니다.

    결과 목록이 표시됩니다. 검색 결과 수는 100개로 제한됩니다.

  4. 검색한 요소를 탐색 트리에 추가하려면 결과 목록에서 원하는 요소를 클릭합니다.

    찾기 창이 닫히고 해당 요소가 탐색 트리의 하위 노드 목록에 추가됩니다.
