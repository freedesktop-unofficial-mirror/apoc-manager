介紹

Desktop Manager 工具提供桌面應用程式之配置設定 (稱為設定檔) 的集中管理。Desktop Manager 可讓您檢視並指定多個設定檔給一個組織階層或一個網域階層的多個元素。當使用者啟動 Desktop Manager 所管理的桌面階段作業或應用程式時，即會套用這些設定檔。它會擷取與使用者或與執行該應用程式的主機相關的所有設定檔，同時其設定也會與應用程式的本機預設值和使用者的自訂設定進行整合。設定檔可用以提供應用程式的一組集中管理的預設值，或用以執行必要設定。

應熟悉的字詞：

  • 配置儲存庫：組織和網域樹狀結構容器，其中會儲存設定檔。

  • 網域：網域樹狀結構中的元素。表示電腦網路中的一個範圍。可能包含網域和主機。

  • 元素：可以指定給配置資料的邏輯物件。使用者、角色/群組和組織均為 Desktop Manager 的實體範例。

  • LDAP：簡易型目錄存取協定。LDAP 為透過 TCP/IP 執行的目錄服務協定。RFC 1777「簡易型目錄存取協定」中定義了 LDAP 的詳細資訊。

  • 組織：組織樹狀結構中的元素。表示組織或子組織；例如，電話客服中心。可能包含組織、使用者或角色。

  • 設定檔：配置設定的具名容器。儲存在配置儲存庫中，並且可以指定給元素。
