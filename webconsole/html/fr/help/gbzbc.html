Suppression d'un référentiel de configuration

Étapes

  1. Sélectionnez la case à cocher correspondant au référentiel de configuration à supprimer.

  2. Cliquez sur le bouton Supprimer.

    Une boîte de dialogue de confirmation s'affiche.

  3. Si vous souhaitez vraiment supprimer le référentiel de configuration, cliquez sur le bouton OK de la boîte de dialogue de confirmation.
