Création d'un référentiel de configuration

Un référentiel de configuration est un emplacement dans lequel sont stockées les informations de hiérarchie organisationnelle ainsi que celles des profils et des assignations de chaque élément.

Étapes

  1. Dans le tableau Référentiels de configuration, cliquez sur Nouveau.

    L'assistant Nouveau référentiel de configuration s'ouvre. Cet assistant sert à créer un référentiel de configuration pour Desktop Manager.

  2. Dans le champ Nom du référentiel, saisissez un nom pour le nouveau référentiel de configuration, puis cliquez sur Suivant.

  3. Dans la liste des types de référentiel, sélectionnez un type.

    L'assistant peut configurer trois types de référentiel :

    • LDAP : le LDAP fournit les informations sur la hiérarchie et les données sont stockées sur ce serveur.

    • Fichier : un fichier fournit les informations sur la hiérarchie et les données sont stockées dans des répertoires et des fichiers.

    • Hybride : le LDAP fournit les informations sur la hiérarchie, mais les données sont stockées dans des répertoires et des fichiers.

  4. Cliquez sur le bouton Suivant.

  5. Saisissez les détails du serveur LDAP à configurer pour ce référentiel.

    Le nom d'hôte et le port sur lequel le serveur s'exécute sont requis. Vous pouvez également choisir de connecter ce serveur via SSL.

  6. Les étapes suivantes dépendent des choix effectués précédemment. Suivez les instructions fournies par l'assistant.
