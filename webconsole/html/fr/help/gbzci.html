Modification du nom d'un référentiel de configuration

Étapes

  1. Sélectionnez la case à cocher correspondant au référentiel de configuration à renommer.

  2. Cliquez sur le bouton Renommer.

    La boîte de dialogue de modification du nom s'affiche.

  3. Entrez le nouveau nom du référentiel de configuration dans le champ de saisie de la boîte de dialogue, puis cliquez sur OK.
