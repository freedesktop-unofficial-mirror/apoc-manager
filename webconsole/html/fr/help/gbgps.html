Importation de profils

Étapes

  1. Vérifiez que les toutes les cases sont cochées.

  2. Choisissez Importer dans la liste des actions.

    La boîte de dialogue Importer un profil s'ouvre.

  3. Spécifiez le chemin d'accès au fichier à importer ou cliquez sur Parcourir pour le choisir dans le sélecteur de fichiers par défaut de votre système d'exploitation.

  4. Cliquez sur le bouton Importer.
