Assignation d'un élément à un profil

Étapes

  1. Cliquez sur le bouton Assigner un élément.

    La fenêtre qui s'ouvre affiche une arborescence de navigation.

  2. Parcourez l'arborescence jusqu'à l'élément souhaité, puis sélectionnez-le.

    Astuce – Si vous n'arrivez pas à localiser l'élément, cliquez sur Recherche.

    L'élément s'affiche ensuite dans le tableau Éléments assignésÂ : il est à présent assigné au profil actuel.
