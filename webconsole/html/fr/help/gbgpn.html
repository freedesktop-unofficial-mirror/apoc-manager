Assignation d'un profil

Étapes

  1. Cliquez sur le bouton Assigner un profil.

    La boîte de dialogue qui s'affiche répertorie les profils qu'il est possible d'assigner à l'élément sélectionné.

    Remarque – Les profils déjà assignés à l'élément sélectionné, ainsi que les profils non stockés dans l'élément ou situés au-dessus de l'élément sont absents de cette liste.

  2. Sélectionnez au moins un profil dans la liste, puis cliquez sur OK.

    Le nouveau profil assigné s'affiche à présent dans le tableau Profils assignés.
