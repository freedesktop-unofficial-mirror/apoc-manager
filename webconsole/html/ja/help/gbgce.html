概要

Desktop Manager は、プロファイルと呼ばれる構成設定の一元管理をデスクトップアプリケーションに提供するツールです。Desktop Manager を使用すると、プロファイルを表示したり、組織やドメインの階層のさまざまな要素にプロファイルを割り当てることができます。プロファイルは、Desktop Manager で管理されているデスクトップセッションやアプリケーションをユーザーが開始したときに適用されます。アプリケーションを実行しているユーザーやホストに関連するプロファイルがすべて取得され、その設定がアプリケーションのローカルデフォルトとユーザーのカスタム設定に統合されます。プロファイルを使用すると、一元管理されるデフォルトセットをアプリケーションに提供したり、必要な設定を強制したりできます。

重要な用語

  • 構成リポジトリ:組織とドメインツリーのコンテナ、およびその中に格納されているプロファイル。

  • ドメイン:ドメインツリー内の要素。コンピュータネットワーク内のレルムを表します。ドメインとホストを含んでいる場合があります。

  • 要素:構成データを割り当てることができる論理オブジェクト。ユーザー、役割/グループ、組織などは Desktop Manager が認識するエンティティーの例です。

  • LDAP:Lightweight Directory Access Protocol。LDAP は TCP/IP 上で動作するディレクトリサービスプロトコルです。LDAP の詳細は、RFC 1777 (The Lightweight Directory Access Protocol) で定義されています。

  • 組織:組織ツリー内の要素。組織や、「コールセンター」などのサブ組織を表します。組織、ユーザー、または役割を含んでいる場合があります。

  • プロファイル:構成の設定用の名前付きのコンテナ。プロファイルは構成リポジトリに格納され、要素に割り当てることができます。
