Configuration Manager Help

Welcome to the Sun Java™ Desktop System Configuration Manager Help pages.

Using the Configuration Manager, administrators can assign profiles to individual users or organisational units stored in a central configuration storage such as an existing corporate LDAP server. The configuration settings defined in these profiles are then applied on behalf of the users to the applications running on the Sun Java Desktop System such as the Gnome desktop, Mozilla, StarOffice and Gnome applications like Evolution.

To get help for a specific part of the Configuration Manager, click on one of the links on the left.
