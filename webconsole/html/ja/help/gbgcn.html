構成リポジトリの新規作成

構成リポジトリは、組織の階層情報と、各要素のプロファイルと割り当てのデータが格納される場所です。

手順

  1. 構成リポジトリテーブルで「新規」ボタンをクリックします。

    構成リポジトリの新規作成ウィザードが開きます。このウィザードは、Desktop Manager で使用する構成リポジトリを作成するために使用します。

  2. 「リポジトリ名」フィールドに、新しい構成リポジトリの名前を入力して、「次へ」をクリックします。

  3. 「リポジトリの種類」リストからリポジトリの種類を選択します。

    ウィザードを使用すると、3 種類のリポジトリを構成できます。

    • LDAP: 階層は LDAP から取得され、データはすべて LDAP に格納されます。

    • ファイルベース: 階層はファイルから取得され、データはすべてディレクトリとファイルに格納されます。

    • ハイブリッド: 階層は LDAP から取得されますが、データはすべてディレクトリとファイルに格納されます。

  4. 「次へ」ボタンをクリックします。

  5. このリポジトリに構成する LDAP サーバーの詳細を入力します。

    サーバーを実行するホスト名とポートが必要になります。また、このサーバーに SSL を使用して接続するかどうかも選択できます。

  6. 以降のステップは、前のステップで選択したオプションによって異なります。ウィザードの説明に従ってください。
