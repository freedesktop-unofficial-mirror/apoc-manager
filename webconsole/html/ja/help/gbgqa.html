「構成リポジトリ」ウィンドウ

構成リポジトリは、アプリケーションの設定に必要な構成データと組織の階層情報が各要素のプロファイルと割り当てデータと一緒に格納されている場所です。 「構成リポジトリ」ウィンドウで、リポジトリの新規作成や、既存のリポジトリの操作や同期ができます。

「構成リポジトリ」ウィンドウは、マストヘッドと構成リポジトリリストで構成されています。

マストヘッドには、一般的なリンクが多数あります。マストヘッドの上部には、 4 つのリンクが並ぶユーティリティーバーがあります。以下に、これらのリンクについて左から順に説明します。

  • 「コンソール」ボタンをクリックすると、Java Web Console の起動ページが開きます。

  • 「バージョン」ボタンをクリックすると、Desktop Manager のバージョン情報を示すウィンドウが開きます。.

  • 「ログアウト」ボタンをクリックすると、Sun Web Console、つまり Configuration Manager からログアウトし、「ログイン」ページに戻ります。

  • 「ヘルプ」ボタンをクリックすると、オンラインヘルプページが開きます。

マストヘッドの下部には、製品名、サーバー名、および現在ログインしている管理者の名前が表示されます。.

「構成リポジトリテーブル」、「構成リポジトリの新規作成」を参照してください。
