Umbenennen von Profilen

Schritte

  1. Wählen Sie das Kontrollkästchen für das umzubenennende Profil aus.

  2. Wählen Sie in der Liste “Aktionen” die Option “Umbenennen”.

    Ein Dialogfeld wird angezeigt.

  3. Geben Sie den neuen Namen für das Profil in das Textfeld des Dialogfelds ein und klicken Sie auf “OK”.
