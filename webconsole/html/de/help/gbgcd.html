Profile — Alle

Auf der Seite “Profile — Alle” werden alle verfügbaren Profile, einschließlich derer, die keinem Element zugewiesen sind, angezeigt. Die Profile werden in zwei Tabellen angezeigt, einer für Domänenprofile und einer für Organisationsprofile.

In den Profiltabellen können Sie die folgenden Aktionen durchführen:

  • Erstellen neuer Profile

  • Löschen eines Profils

  • Umbenennen von Profilen

  • Kopieren und Verschieben von Profilen

  • Importieren von Profilen

  • Exportieren von Profilen
