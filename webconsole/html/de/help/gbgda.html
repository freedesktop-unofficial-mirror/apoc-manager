Einstellungsübersicht

Auf der Seite “Einstellungsübersicht” werden alle Konfigurationseinstellungen angezeigt, die Daten für das ausgewählte Profil enthalten, können jedoch nicht geändert werden.

Die Seite “Einstellungsübersicht” enthält die gleichnamige Tabelle. Diese Tabelle zeigt die Konfigurationseinstellungen, die Daten für das aktuelle Profil enthalten, in Kategorien gruppiert an. Die Tabelle hat die folgenden Felder:

  • Name: Der Name der Einstellung

  • Wert: Der Wert der Einstellung

  • Status: Die Statusspalte enthält Einträge der Art “Definiert in <Profilname>”, wobei <Profilname> ein Link ist, mit dem Sie die Seite “Konfigurationseinstellungen” des Profileditors aufrufen und zur entsprechenden Einstellung innerhalb dieses Profils wechseln können.
