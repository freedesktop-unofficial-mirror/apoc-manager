Aufheben der Zuweisung von Profilen

Schritte

  1. Wählen Sie das Kontrollkästchen für das Profil aus, dessen Zuweisung Sie aufheben möchten.

  2. Klicken Sie auf die Schaltfläche “Profilzuweisung aufheben”.

    Das Profil ist nun nicht mehr zugewiesen und wird nicht mehr in der Tabelle “Zugewiesene Profile” angezeigt.

    Warnung – Dieser Vorgang wird ohne vorherigen Warndialog durchgeführt. Haben Sie versehentlich eine Profilzuweisung aufgehoben, können Sie die Zuweisung mithilfe der Funktion “Profil zuweisen” wiederherstellen.
