Zuweisen von Elementen zu Profilen

Schritte

  1. Klicken Sie auf die Schaltfläche “Element zuweisen”.

    Ein Fenster mit einer Navigationsstruktur wird angezeigt.

  2. Wählen Sie das gewünschte Element in der Struktur aus.

    Tipp – Wenn Sie das gewünschte Element nicht finden können, klicken Sie auf die Schaltfläche “Suchen”, um das Fenster “Suchen” zu öffnen, in dem Sie das Element suchen können.

    Das Element wird nun in der Tabelle “Zugewiesene Elemente” als dem aktuellen Profil zugewiesen angezeigt.
