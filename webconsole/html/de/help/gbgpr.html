Löschen von Profilen

Schritte

  1. Aktivieren Sie das Kontrollkästchen für das zu löschende Profil.

    Ein Bestätigungsdialogfeld wird geöffnet.

    Achtung – Beachten Sie, dass mit dem Befehl "Löschen" alle Zuweisungen entfernt werden und danach das Profil selbst gelöscht wird. Danach besteht keine Möglichkeit mehr, es wiederherzustellen. Wenn Sie ein Profil löschen, werden auch Zuweisungen zu anderen Elementen als dem aktuellen Element entfernt. Wenn Sie nur die Profilzuweisung aus einem Element entfernen möchten, verwenden Sie "Zuweisung aufheben".

  2. Klicken Sie auf die Schaltfläche "Löschen".

  3. Wenn Sie das Profil wirklich löschen möchten, klicken Sie im Bestätigungsdialogfeld auf "OK".
