Fenster “Konfigurationsdatensammlungen”

In Konfigurationsdatensammlungen werden die Konfigurationsdaten gespeichert, die Sie für die Konfiguration Ihrer Anwendungen benötigen. Außerdem werden hier Informationen zur Organisationshierarchie und zu den Profil- und Zuweisungsdaten für jedes Element gespeichert. Im Fenster “Konfigurationsdatensammlungen” können Sie neue Datensammlungen erstellen sowie mit vorhandenen Datensammlungen arbeiten und diese synchronisieren.

Das Fenster “Konfigurationsdatensammlungen” besteht aus einem Titelrahmen und der Liste “Konfigurationsdatensammlungen”.

Darüber hinaus enthält der Titelrahmen (auch “Impressum” genannt) einige allgemeine Links: Der obere Teil des Titelrahmens enthält eine Leiste mit vier Links. Diese sind (von link nach rechts):

  • Konsolenschaltfläche zum Öffnen der Java Web Console-Startseite

  • Versionsschaltfläche zum Öffnen eines Fensters mit Versionsinformationen zu Desktop Manager

  • Abmeldeschaltfläche zum Abmelden von Java Web Console und somit auch von Configuration Manager (Anmeldeseite wird angezeigt)

  • Hilfeschaltfläche zum Öffnen der Online-Hilfe

Im unteren Bereich des Titelrahmens werden der Name des Produkts, des Servers und des derzeit angemeldeten Administrators angezeigt.
