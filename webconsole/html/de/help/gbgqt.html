Verwenden der Funktion “Suchen”

Mit der Funktion “Suchen” können Sie nach allen Elementen suchen, die dem erweiterten Element direkt untergeordnet sind. Diese Funktion verhält sich anders als die klassische Suchfunktion, die rekursiv nach Elementen sucht.

Schritte

  1. Klicken Sie in der Baumstruktur für die Navigation auf den Knoten “Einige Elemente werden nicht angezeigt”.

    Das Fenster “Suchen” wird geöffnet.

  2. Geben Sie im Feld “Suchen” den Suchbegriff ein.

    Die Suche unterscheidet nicht nach Groß- und Kleinschreibung und ist nicht streng. Das heißt beispielsweise, dass für den Suchbegriff “bc” die Elemente “ABC” und “bcd” gefunden werden. Ein Asterisk (*) stellt eine beliebige Zeichenfolge in einem Suchbegriff dar. Der Suchbegriff kann mehrere Asteriske enthalten und ihre Position innerhalb des Suchbegriffs ist frei wählbar. Ein leeres Feld entspricht dem Suchbegriff “*”.

  3. Klicken Sie auf die Schaltfläche “Suchen”.

    Eine Ergebnisliste wird angezeigt. Es werden maximal 100 Treffer angezeigt.

  4. Wenn Sie der Baumstruktur für die Navigation ein gefundenes Element hinzufügen möchten, klicken Sie im Suchergebnis auf das gewünschte Element.

    Das Fenster “Suchen” wird geschlossen und das Element wird der Liste von Teilknoten in der Baumstruktur für die Navigation hinzugefügt.
