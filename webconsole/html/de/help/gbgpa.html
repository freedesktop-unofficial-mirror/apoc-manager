Suchen nach Elementen

Mit der Suchfunktion können Sie eine rekursive Suche nach einem Element in der Domänen- oder Organisationshierarchie durchführen. Es werden nicht nur die direkt untergeordneten Elemente eines Elements gesucht, sondern alle untergeordneten Elemente. Es steht außerdem eine Suchfunktion zur Verfügung, mit der Sie ausschließlich nach Elementen suchen können, die einem anderen Element direkt untergeordnet sind.

Bei einer Suche wird das Fenster “Suchen” geöffnet, in dem Sie eine erweiterte Suche durchführen können.

Schritte

  1. Geben Sie in das Suchfeld den Namen beziehungsweise einen Teil des Namens eines Elements ein, nach dem Sie suchen möchten.

    Hinweis – Die Suche unterscheidet nicht nach Groß- und Kleinschreibung und ist nicht streng. Das heißt beispielsweise, dass für den Suchbegriff “bc” die Elemente “ABC” und “bcd” gefunden werden. Ein Asterisk (*) stellt eine beliebige Zeichenfolge in einem Suchbegriff dar. Der Suchbegriff kann mehrere Asteriske enthalten und ihre Position innerhalb des Suchbegriffs ist frei wählbar. Ein leeres Feld entspricht dem Suchbegriff “*”.

  2. Klicken Sie auf die Schaltfläche “Suchen”.

    Das Fenster "Suchen" wird aufgerufen. Eine in das Suchfeld des Navigationsbereichs eingegebene Zeichenfolge wird in das Suchfeld des Fensters “Suchen” übernommen und es wird automatisch eine Suche nach diesem Suchbegriff durchgeführt. Die Treffer der Suche werden in der Ergebnistabelle aufgeführt, die den Namen, Typ und Pfad aller gefundenen Elemente anzeigt.

    Hinweis – Es werden maximal 100 Suchergebnisse angezeigt.

    Wurden keine Ergebnisse gefunden oder haben Sie das Suchfeld im Navigationsbereich leer gelassen, können Sie im Suchfeld des Fensters “Suchen” den Namen eines Elements eingeben.

    Tipp – Verwenden Sie die Liste zum Ändern des Typs, die sich neben dem Suchfeld befindet, um nach Elementen eines bestimmten Typs (beispielsweise “Domänen suchen”) zu suchen.

  3. Bei Bedarf können Sie auf die Schaltfläche “Erweiterte Optionen anzeigen” klicken, um der Suche erweiterte Optionen hinzuzufügen.

    Folgende erweiterte Optionen stehen zur Verfügung:

    • Beschränken auf: Hier können Sie angeben, bei welchem Element der Suchalgorithmus die Suche im Baum beginnen soll.

    • Angezeigte Ergebnisse pro Seite: Hier können Sie festlegen, wie viele Ergebnisse auf jeder Seite der Ergebnistabelle angezeigt werden sollen.

  4. Klicken Sie erneut auf die Schaltfläche “Suchen”, um die Ergebnisse einer neuen oder geänderten Suche anzuzeigen.

    Hinweis – Durch Klicken auf die Schaltfläche “Zurücksetzen” werden alle Suchparameter auf ihre Vorgaben zurückgesetzt und ein gegebenenfalls im Suchfeld eingegebener Suchbegriff wird gelöscht.
