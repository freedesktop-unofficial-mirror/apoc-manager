Anzeigen und Bearbeiten der Konfigurationseinstellungen von Profilen

Schritte

  1. Um eine bestimmte Konfigurationseinstellung eines Profils anzuzeigen, können Sie sie im rechten Bereich der Seite in der Kategorienliste auswählen.

    Die Konfigurationseinstellungen sind auf hierarchische Weise in Kategorien unterteilt, die direkt den Konfigurationsoptionen von Desktopanwendungen wie StarOffice™, Mozilla™, Evolution oder Gnome entsprechen.

  2. Klicken Sie auf die gewünschte Konfigurationseinstellung.

    Die Einstellungen des ausgewählten Profils werden angezeigt, die alle verfügbaren Einstellungen für dieses Profil angeben. Bei Bedarf werden zusätzliche Kontextinformationen, beispielsweise zu den Auswirkungen einzelner Einstellungen oder zu den zulässigen Werten, angegeben.

    Hinweis – Wenn Sie statt einer Konfigurationseinstellung eine Kategorie auswählen, wird eine Liste mit den Einstellungen oder Unterkategorien dieser Kategorie angezeigt.

  3. Sie können für jede Konfigurationseinstellung entweder einen bestimmten Wert angeben oder die Einstellung nicht definieren.

    Definieren Sie eine Einstellung nicht, ist die Anwendung nicht von dieser Profileinstellung betroffen und verwendet den anwendungseigenen Vorgabewert. Im Allgemeinen zeigt der Profileditor den Vorgabewert einer Anwendung in einem zusätzlichen Hilfetext oder - falls möglich - direkt in der Optionsliste der Einstellung an.

    Hinweis – Beachten Sie stets, dass mehrere Profile Auswirkungen auf denselben Benutzer haben können. Beispiel: Sie erstellen ein Profil mit Einstellungen für eine bestimmte Organisation. Diese Einstellungen stehen automatisch allen Benutzern zur Verfügung, die Teil dieser Organisation sind. Ähnlich verhält es sich, wenn Sie zusätzliche Profile mit Konfigurationseinstellungen für Unterorganisationen definieren. Diese können Einstellungen der übergeordneten Organisation überschreiben (beispielsweise wenn für Unterorganisationen andere Proxy-Einstellungen festgelegt wurden), andere Einstellungen jedoch unberührt lassen. Für einen bestimmten Benutzer würden in diesem Fall Einstellungen gelten, die das Ergebnis einer Zusammenführung wären, die alle der Organisation, den Unterorganisationen und möglicherweise dem Benutzer zugewiesenen Profile einschließt.

    Tipp – In einigen Fällen ist es ratsam, diese Zusammenführung zu steuern und eine bestimmte Konfigurationseinstellung zu erzwingen, sodass andere Profile diese Einstellung nicht überschreiben und einen anderen Wert definieren können. Im Profileditor können Sie für jede Einstellung das gewünschte Zusammenführungsverhalten auswählen. Die Standardeinstellung ist, dass andere Profile eine Einstellung überschreiben können. Wenn Sie diese Option deaktivieren, wird die Einstellung erzwungen, das heißt andere Profile können den von Ihnen festgelegten Wert nicht überschreiben.
