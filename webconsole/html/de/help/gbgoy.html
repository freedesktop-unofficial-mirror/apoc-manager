Inhaltsbereich

Wenn Sie ein Element auswählen, werden Informationen zu diesem Element im Inhaltsbereich der Seite “Profile — Baumansicht” angezeigt. Der Inhaltsbereich kann bis zu drei Tabellen enthalten:

  • Tabelle “Zugewiesene Profile”: Diese Tabelle wird immer angezeigt. Sie zeigt die Profile an, die dem derzeit ausgewählten Element zugewiesen sind. Diese Tabelle hat die drei Spalten “Name”, “Autor” und “ Letzte Änderung am”. Der Wert der Spalte “ Letzte Änderung am” ändert sich nur, wenn eine Einstellung des Profils geändert wird, nicht aber, wenn das Profil selbst umbenannt oder verschoben wird bzw. wenn dessen Priorität geändert wird.

    In der Tabelle “Zugewiesene Profile” können Sie die folgenden Aktionen durchführen:

    • Erstellen neuer Profile

    • Löschen eines Profils

    • Zuweisen von Profilen

    • Aufheben der Zuweisung von Profilen

    • Umbenennen von Profilen

    • Kopieren und Verschieben von Profilen

    • Importieren von Profilen

    • Exportieren von Profilen

  • Tabelle “Übernommene Profile”: Diese Tabelle wird angezeigt, wenn einem der Elemente, die in der Hierarchie oberhalb des ausgewählten Elements sind, ein Profil zugewiesen ist. Dieses Profil wird in der Tabelle aufgeführt. Die Tabelle enthält die beiden Spalten “Name” und “Zugewiesen”. In der Spalte “Zugewiesen” werden der Pfad und der Name des Elements angezeigt, dem das Profil zugewiesen ist.

  • Tabelle “Benutzer” oder “Rollen”: Diese Tabelle wird angezeigt, wenn ein Element vom Typ “Benutzer” oder “Rolle” ausgewählt ist. Handelt es sich bei einem Element um einen Benutzer, der Mitglied mindestens einer Rolle ist, werden diese Rollen in der Tabelle “Rollen” angezeigt. Handelt es sich bei einem Element um eine Rolle mit Mitgliedern, werden diese Mitglieder in der Tabelle “Benutzer” angezeigt.
