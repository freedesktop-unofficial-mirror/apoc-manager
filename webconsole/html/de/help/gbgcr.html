Erstellen neuer Profile

Schritte

  1. Klicken Sie in der gewünschten Profiltabelle (“Organisation” oder “Domäne”) auf die Schaltfläche “Neu”.

    Der Profileditor wird geöffnet.

  2. Befolgen Sie die Anweisungen.

    Weitere Informationen zum Erstellen neuer Profile mit dem Profileditor finden Sie im Abschnitt Profileditor der Online-Hilfe.
