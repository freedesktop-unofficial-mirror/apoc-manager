Exportieren von Profilen

Schritte

  1. Wählen Sie das Kontrollkästchen für das zu exportierende Profil aus.

  2. Wählen Sie in der Liste “Aktionen” die Option “Exportieren”.

    Ein Dialogfeld wird geöffnet, in dem Sie auswählen können, wo die exportierte Datei gespeichert werden soll.

  3. Wählen Sie die Option zum Speichern auf einem Datenträger und klicken Sie auf “OK”.

  4. Wählen Sie den Speicherort für die exportierte Datei aus und klicken Sie auf “OK”.
