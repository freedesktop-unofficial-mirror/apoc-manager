Ändern des Speicherorts für “Anwendung von”

Schritte

  1. Klicken Sie auf die Schaltfläche “Ändern”.

    Ein Fenster mit einer Baumstruktur für die Navigation wird angezeigt.

  2. Wählen Sie in der Baumstruktur das gewünschte Element aus.

    Tipp – Wenn Sie das gewünschte Element nicht finden können, klicken Sie auf die Schaltfläche “Suchen”, um das Fenster “Suchen” zu öffnen, indem Sie das Element suchen können.

  3. Klicken Sie auf die Schaltfläche “OK”.
