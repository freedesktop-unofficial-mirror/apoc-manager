Festlegen der allgemeinen Eigenschaften für ein Profil

Schritte

  1. Geben Sie im Feld “Name” den gewünschten Namen für das Profil ein.

  2. Geben Sie bei Bedarf im Feld “Kommentar” einen Kommentar ein.

    Sie können dieses Feld auch leer lassen.

    Hinweis – Die Felder “Autor” und “Letzte Änderung am” sind schreibgeschützt und können nicht geändert werden. Im Feld “Autor” wird der Name des Profilautors und im Feld “Letzte Änderung am” werden Datum und Uhrzeit der letzten Änderung des Profils angezeigt.

  3. Klicken Sie auf die Schaltfläche “Speichern”.

    Wenn Sie das Fenster schließen, ohne neu eingegebene Informationen gespeichert zu haben, wird ein Warndialog angezeigt, der Sie zum Speichern der Einstellungen auffordert.
