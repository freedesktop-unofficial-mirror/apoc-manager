Erstellen neuer Konfigurationsdatensammlungen

Konfigurationsdatensammlungen sind Orte, an denen Organisationshierarchieinformationen zusammen mit den Profil- und Zuweisungsdaten für jedes Element gespeichert werden.

Schritte

  1. Klicken Sie in der Tabelle “Konfigurationsdatensammlungen” auf die Schaltfläche “Neu”.

    Der Assistent für neue Konfigurationsdatensammlungen wird geöffnet. Mit diesem Assistenten können Sie eine Konfigurationsdatensammlung für die Verwendung mit Desktop Manager erstellen.

  2. Geben Sie im Feld “Name der Datensammlung” einen Namen für die neue Konfigurationsdatensammlung ein und klicken Sie auf “Weiter”.

  3. Wählen Sie in der Liste “Typ der Datensammlung” den Typ der Datensammlung aus.

    Mit dem Assistenten können drei Arten von Datensammlungen erstellt werden:

    • LDAP: Die Hierarchie wird LDAP entnommen und alle Daten werden in LDAP gespeichert.

    • Dateibasiert: Die Hierarchie wird einer Datei entnommen und alle Daten werden in Verzeichnissen und Dateien gespeichert.

    • Kombiniert: Die Hierarchie wird LDAP entnommen, aber alle Daten werden in Verzeichnissen und Dateien gespeichert.

  4. Klicken Sie auf die Schaltfläche “Weiter”.

  5. Geben Sie die Details des LDAP-Servers ein, den Sie für diese Datensammlung konfigurieren möchten.

    Sie müssen den Hostnamen und den Port des Servers eingeben. Außerdem können Sie auswählen, ob für die Verbindung zum Server SSL verwendet werden soll.

  6. Je nach ausgewählten Optionen müssen möglicherweise weitere Angaben gemacht werden. Befolgen Sie die Anweisungen.
