Zugewiesene Elemente

Auf der Seite “Zugewiesene Elemente” werden die Elemente aufgeführt, die dem aktuellen Profil zugewiesen sind. Hier können Sie Elemente zuweisen und die Zuweisung von Elementen aufheben.

In der Tabelle “Zugewiesene Elemente” werden die Elemente des aktuellen Profils angezeigt. Sie enthält folgende Informationen:

  • Name: Name des Elements, dem das aktuelle Profil zugewiesen ist.

  • Pfad: Speicherort des Elements, dem das aktuelle Profil zugewiesen ist.

Siehe Zuweisen von Elementen zu Profilen, Aufheben der Zuweisung von Elementen zu Profilen
