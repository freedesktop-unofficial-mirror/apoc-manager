Erweiterte Optionen

Auf der Seite “Erweiterte Optionen” können Sie die erweiterten Optionen für das aktuelle Profil festlegen. Auf dieser Seite stehen zwei Hauptfunktionen zur Verfügung:

  • Anwendbar von: Ermöglicht das Verschieben des Profils an einen anderen Speicherort im Organisations- oder Domänenbaum. Durch Verschieben des Speicherorts ändern Sie auch den Satz von Elementen, denen das Profil zugewiesen beziehungsweise auf die es angewendet werden kann. Dies liegt daran, dass ein Profil nur dem Element zugewiesen werden kann, an dem es gespeichert werden kann, beziehungsweise den Elementen zugewiesen werden kann, die sich unterhalb des Elements befinden, an dem es gespeichert ist.

    Warnung – Es ist nicht möglich, den Parameter “Anwendung von” so zu ändern, dass das daraus resultierende Layout gegen die oben erwähnte Regel verstößt. Dies könnte beispielsweise passieren, wenn Sie ein Profil in der Hierarchie nach unten verschieben und eine vorhandene Zuweisung, die das Profil mit einem Element unterhalb des Speicherorts des Profils verbindet, das Profil mit demselben Element verbinden würde, nachdem das Element verschoben wurde. Nun befindet sich das Element jedoch oberhalb des Speicherorts des Profils. Versuchen Sie dies, wird der Vorgang abgebrochen und eine Warnmeldung wird angezeigt.

  • Zusammenführungsreihenfolge: Gibt an, in welcher Reihenfolge alle an demselben Element gespeicherten Profile zusammengeführt werden.

    Da einem einzelnen Element mehrere Profile zugewiesen werden können und mehrere Profile dieselbe Einstellung festlegen können, muss die Reihenfolge der Zusammenführung der Profile für ein einzelnes Element bestimmt werden. Auf diese Weise werden Konflikte behoben, wenn eine Einstellung in zwei oder mehr Profilen definiert ist, die demselben Element zugewiesen sind.

    Der Speicherort wird als Erstes zur Bestimmung der Zusammenführungsreihenfolge herangezogen. Das Profil, das an einem Element gespeichert ist, das näher an der Root des Baums ist, wird als Erstes zusammengeführt. Das tiefer in der Baumhierarchie gespeicherte Profil wird folglich danach gespeichert. Die Einstellungen eines später zusammengführten Profils überschreiben die Einstellungen eines früher zusammengeführten Profils.
